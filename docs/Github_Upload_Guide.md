# Загрузка проекта на GitHub через Visual Studio Code

Есть два основных способа загрузить (опубликовать) ваш проект на GitHub напрямую из VS Code: через графический интерфейс (GUI) или через встроенный терминал.

## Способ 1: Использование графического интерфейса VS Code (Рекомендуется для новичков)

1. **Откройте вкладку Source Control (Управление версиями):**
   - Нажмите на иконку с тремя узлами на левой панели (или нажмите `Ctrl+Shift+G` / `Cmd+Shift+G` на Mac).

2. **Инициализируйте репозиторий:**
   - Если репозиторий еще не создан, вы увидите кнопку **"Initialize Repository"** (Инициализировать репозиторий) или **"Publish to GitHub"** (Опубликовать на GitHub). Нажмите на неё.

3. **Сделайте первый коммит (если нужно):**
   - Введите сообщение для коммита (например, `Initial commit`) в текстовое поле над списком файлов.
   - Нажмите кнопку **"Commit"** (или галочку сверху). Вас могут попросить добавить файлы в индекс (stage) — согласитесь, выбрав "Yes".

4. **Опубликуйте ветку на GitHub:**
   - После коммита появится кнопка **"Publish Branch"** (Опубликовать ветку). Нажмите её.
   - VS Code предложит вам выбрать:
     - *Publish to GitHub public repository* (публичный репозиторий)
     - *Publish to GitHub private repository* (приватный репозиторий)
   - Выберите нужный вариант. Название репозитория по умолчанию будет совпадать с названием вашей папки.
   - Если вы еще не авторизованы в GitHub через VS Code, появится окно браузера для авторизации.
   - После завершения процесса в правом нижнем углу появится уведомление об успешной загрузке с кнопкой **"Open in GitHub"**.

---

## Способ 2: Использование встроенного терминала (Для продвинутых пользователей)

Если вы предпочитаете работать с командами (или если используете GitHub CLI), выполните следующие шаги:

1. **Откройте терминал в VS Code:**
   - В верхнем меню выберите `Terminal` -> `New Terminal` (или нажмите `Ctrl+~` / `Cmd+~`).

2. **Инициализируйте локальный Git-репозиторий:**
   ```bash
   git init
   ```

3. **Добавьте все файлы в индекс:**
   ```bash
   git add .
   ```

4. **Создайте коммит:**
   ```bash
   git commit -m "Initial commit"
   ```

5. **Создайте репозиторий на GitHub:**
   - Зайдите на сайт GitHub и создайте новый пустой репозиторий (без README, .gitignore или лицензии).
   - Скопируйте ссылку на этот репозиторий (например, `https://github.com/ВашЛогин/ВашРепозиторий.git`).

6. **Привяжите удаленный репозиторий:**
   ```bash
   git remote add origin https://github.com/ВашЛогин/ВашРепозиторий.git
   ```

7. **Измените название основной ветки на `main` (если требуется):**
   ```bash
   git branch -M main
   ```

8. **Отправьте код на GitHub:**
   ```bash
   git push -u origin main
   ```

> **Примечание:** Если у вас установлен **GitHub CLI** (`gh`), вы можете создать репозиторий и загрузить код одной командой прямо из терминала:
> ```bash
> gh repo create НазваниеРепозитория --public --source=. --remote=origin --push
> ```
